1. Clone the repository:
```bash
git clone https://github.com/dibya-22/Blinklink.git
```

2. Install dependencies:
```bash
cd Blinklink
npm install
```

3. Create a `.env.local` file in the root directory:
```env

```

4. Run the development server:
```bash
npm run dev
